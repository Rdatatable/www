data.table offers fast

Aggregation

DT[, list(sx=sum(x)), by=z]

Ordered joins

DT1[DT2, nomatch=, mult=]

Add/modify/del columns by reference

DT1[, `:=`(x=val, y=new_val, z=NULL)]

File reading

fread("big_file.csv")

Rolling joins

DT1[DT2, nomatch=, mult=, roll=]

Chained queries

DT1[, cy := cumsum(y), by=x][DT2]

and more, all in a natural syntax: DT[where, select|update, by]
